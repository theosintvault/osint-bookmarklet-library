# OSINT Bookmarklet Library

A curated collection of JavaScript-based bookmarklets designed for rapid, on-page intelligence gathering. These tools function directly within the browser's bookmark bar, allowing investigators to extract data and pivot to external databases without leaving the target page.

### [Live Library Link](https://theosintvault.io/osint-bookmarklet-library)

### Available Bookmarklets
*   **Metadata Extraction:** Extract image EXIF data and page headers.
*   **Social Media Analysis:** Rapid profile ID retrieval and timestamp conversion for major platforms.
*   **Search Pivots:** Highlight text on any page and instantly launch targeted searches across the OSINT Vault ecosystem.
*   **Archiving:** Quick-send URLs to web archives and forensic preservation tools.

### Installation & Use
1. Drag the desired bookmarklet from the library page to your browser's bookmarks bar.
2. Navigate to a target website.
3. Click the bookmarklet to execute the script on the current page.

### Technical Notes
These scripts execute client-side within the context of the active tab. No data is transmitted to The OSINT Vault servers. Users should review individual script logic for specific platform compatibility.
