# Bookmarklet Library

A collection of browser-native JavaScript utilities for instant, on-page metadata extraction and entity pivoting.

[Open Library](https://theosintvault.io)

## Core Capabilities
* On-Page Extraction: Rapidly pull metadata from social profiles and public records without leaving the page.
* Pivoting Logic: Instant transitions from on-page usernames to external lookup engines.
* Zero-Footprint: Lightweight scripts that execute in the browser session with no external dependencies.

## Technical Specifications
* Language: JavaScript (ES6+)
* Deployment: Drag-and-drop browser utility
